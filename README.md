# Melbourne House Price Prediction using Machine Learning

## Overview
This project predicts housing prices in Melbourne using machine learning regression models.  
It includes data preprocessing, feature engineering, and comparison of multiple regression techniques.

---

## Dataset
Melbourne Housing dataset containing features such as:
- Suburb
- Rooms
- Type
- Distance
- Landsize
- BuildingArea
- Property details

---

## Workflow

### Data Preprocessing
- Selected important features
- Handled missing values (mean / zero filling)
- Removed remaining nulls
- Applied one-hot encoding for categorical variables

### Models Used
- Linear Regression
- Lasso Regression
- Ridge Regression

---

## Evaluation
Models are evaluated using R² Score on:
- Training data
- Testing data

- pandas
